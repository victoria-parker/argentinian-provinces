Argentinian Provinces API

This is an API I built to practice Express. When you make a request it returns an array with information of the Argentinian provinces. The information it returns is the name of the province, its capital, its year of foundation and its land area.

To make a request you need to make a get request to https://argentinian-provinces.herokuapp.com/api

If you'd like to get information about a specific province you can make your request to /api/< name of your province > . This will return the object with the data of that specific province.
